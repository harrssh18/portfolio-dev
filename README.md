# Harsh Dubey Portfolio

Personal portfolio for Harsh Dubey, Software Engineer.

## Features

- Dark responsive portfolio layout
- Smooth scroll navigation
- Typed data layer for projects, skills, experience, and links
- Project cards with impact points and tech stack tags
- Timeline-based experience section
- Subtle motion with reduced-motion support

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Content

Portfolio content is managed from:

```text
src/data/portfolio.ts
```

## Deployment

This project is ready for Vercel:

1. Push the repository to GitHub.
2. Import the repository in Vercel.
3. Keep the default Vite settings.
4. Enable automatic deployments from the main branch.
